# leetcode_analysis
leetcode题解整理
## 1 序言
目的是想通过**leetcode**来整理数据结构和算法，以一种**层次清晰、重点突出、易于理解**的方式呈现给大家，一起**hello world，一起改变世界**
## 2 思路
对数据结构和算法分类做**专题**，包括两大部分：
+ 理论
+ 场景
+ 示例（leetcode题及对应的***c++***语言的题解）
