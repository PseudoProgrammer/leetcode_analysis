# 简介
栈（stack）又名堆栈，它是一种运算受限的线性表。限定仅在**表尾进行插入和删除**操作的线性表。这一端被称为栈顶，相对地，把另一端称为栈底
## 特点及用途
**后进先出**，与之相反的是队列（先进先出）
1. 栈内存。
+ 问题：顺序式编程的问题在于很多临时变量的内存未释放，浪费内存资源
+ 方案：函数B相比前面的代码段A，内存后申请先释放
