# 理论
## 深度优先遍历
从根节点出发，沿着左子树方向进行纵向遍历，直到找到叶子节点为止。然后回溯到前一个节点，进行右子树节点的遍历，直到遍历完所有可达节点为止。
## 广度优先遍历
从根节点出发，在横向遍历二叉树层段节点的基础上纵向遍历二叉树的层次。
